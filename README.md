# arrhythmia-classification
ECG signal classification using CNN and hybrid LSTM-CNN architectures. Preprocesses MIT-BIH format data across 5 arrhythmia classes (N, S, V, F, Q), tunes classification thresholds to analyse sensitivity–specificity trade-offs, and compares model performance under class imbalance conditions using Accuracy, Sensitivity, Specificity, and F1-score.
